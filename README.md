# SpectaclesDetection

Model to detect, whether there are spectacles on photo, or not.

Dependencies:
  opencv, 
  numpy, 
  matplotlib, 
  imutils, 
  skimage,  
  scipy, 
  sklearn,
  pickle,
  dlib,
  tqdm,
  keras


Usage:
	python inference.py --test_dir="path to directory with images"
